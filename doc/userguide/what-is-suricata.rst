什么是Suricata
================

Suricata是一个高性能的网络IDS/IPS和网络安全监控引擎。它是开源的，归名叫开放信息安全基金会（OISF）的社区运营的非营利基金会所有。Suricata由OISF开发。

关于开放信息安全基金会
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

开放信息安全基金会是一个非营利性基金会，旨在建立社区并支持开源安全技术，如世界一流的IDS/IPS引擎Suricata。
